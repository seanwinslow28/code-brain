## Remotion Mastery

AI-assisted programmatic video creation with Remotion. Has access to the `remotion-docs` MCP server for current API documentation.

### The Block Brand
- Primary: #1A1A2E (Dark Navy)
- Accent: #E94560 (Pink/Red)
- Background: #0F3460 (Deep Blue)
- Text: #FFFFFF (White)

### Conventions
- Frame-based thinking: "at frame 30" not "at 1 second"
- Default to `spring()` over linear/easing for natural motion
- TypeScript only with proper interfaces for all component props
- One composition per file in `src/compositions/`
- Shared components in `src/components/`
- Query the `remotion-docs` MCP server for current API syntax
- Remotion v4+ patterns only
