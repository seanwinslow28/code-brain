## Advanced Techniques

Power-user productivity techniques for getting the most out of Claude Code.

### Key Conventions
- Use `/compact` when context feels stale or responses degrade
- Use Plan Mode (double `Shift+Tab`) for architectural decisions
- Use Extended Thinking (single `Tab`) for complex reasoning within a task
- Use `.tmp` files for intermediate work that shouldn't persist
- Run multiple Claude Code instances with `git worktree` for parallel development
- Use `systematic-debugging` for four-phase root cause analysis before any fix
- Use `verification-before-completion` to ensure fresh evidence before claiming "done"
