## Master Designer

Design excellence, animation libraries, and premium UI production.

### Conventions
- Default animation library: Motion (Framer Motion)
- Default spring: `stiffness: 100, damping: 30` (Smooth)
- Motion vocabulary: Bouncy (300/10), Smooth (100/30), Snappy (500/30), Heavy (50/20)
- Accessibility: `prefers-reduced-motion` respected on all animations
- Tailwind-first for styling, custom CSS only when Tailwind can't express it
- Mobile-first responsive design
