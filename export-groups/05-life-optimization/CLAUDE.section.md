## Life Optimization

Personal life automation skills. All data stays local. Python scripts for processing, CSV/JSON for storage.

### Available Skills
- **budget-entry**: Quick expense categorization and budget tracking ("log expense: $45 groceries")

### Conventions
- Privacy-first: No cloud APIs for personal data unless explicitly requested
- Python 3.10+ for automation scripts
- CSV and JSON for data storage (no databases for personal data)
- All financial amounts in USD unless specified
