# HEARTBEAT — Job Hunt 2026

## Daily Rhythm

- **Pre-8:30 AM:** Gym. Off-domain.
- **First hour (8:30–9:30 AM) — Sacred Learning Block:** Articles, YouTube, podcasts on AI / PM craft. NOT email, NOT deep work, NOT applications. The "eat and learn" hour to stay sharp on AI PM trends. This is the hardest rule.
- **9:30 AM – 1:00 PM:** Locked-in job-hunt work — parallel activities (Track-C building, searching, applying, outreach). No fixed time slots; the day flexes to whatever is hot.
- **1:00–2:30 PM:** 1–2 hr break (lunch + reset).
- **2:30–5:30 PM:** Continuation of locked-in window — same parallel mode.
- **5:30 PM:** Hard stop on active job-hunt admin (applications, outreach, recruiter replies).
- **Post-5:30 PM:** Informal learning bleed allowed — reading, podcasts, dabbling on Claude Code side projects. Not enforced as work.

- **Portfolio additions:** Continuous, but expect at least one shippable artifact (MCP feature, Substack post, resume update) per month.
- **AI trend audit:** Monthly check — what's new in the AI PM space and how does it map onto Sean's portfolio?
- **Admin milestones:** Severance signing window (~2026-05-11), UI recertification cadence (TBD post-filing), COBRA / ACA SEP deadline 2026-07-03, monthly target-list reshuffle.

## Seasonal Shifts (8-week sprint)

- **Weeks 1–2 (2026-05-04 → 2026-05-17):** Runway + positioning artifacts. Heavy admin (severance, UI, COBRA, resume v1, LinkedIn refresh, GitHub README, MCP v0 scaffolding). Deep-work weighted.
- **Weeks 3–4 (2026-05-18 → 2026-05-31):** Applications go hot. Pipeline target = 5 quality apps/week. Outreach volume peaks. Building continues.
- **Weeks 5–6 (2026-06-01 → 2026-06-14):** Interview loops + Track-C MCP server v1 ship. Interview-prep blocks displace some application volume. Building / showcase work emphasized.
- **Weeks 7–8 (2026-06-15 → 2026-06-28):** Offers + decisions. Outreach tapers, decision-making and negotiation prep dominate.
- **Through-line across all phases:** Lean toward more building / showcase time, not less. Track-C is the differentiator.

- **5:30 PM hard stop** on active job-hunt admin. Informal learning after is not work and is not enforced.
- **Friday retro** is the only mandatory ritual — self + Claude review of the week's pipeline + qualitative wins/misses.
