# Operating Model — Job Hunt 2026

## TL;DR

Sean is running an 8-week post-Block job hunt anchored on three parallel tracks (Runway / Pipeline / Differentiator), with the differentiator — a public MCP server — as the protected portfolio artifact. He works in one big fluid 8:30 AM–5:30 PM container with a sacred learning hour up front, a mandatory mid-day break, and a Friday retro as the only non-negotiable ritual. Agents draft and discover; Sean owns every word that hits another human's inbox.

## Identity in This Domain

- **Role:** Job seeker (after 2026-05-04 layoff from The Block, delivered by Larry Cermak + Vicky Lu).
- **Targets, in priority order:** AI PM > Tech PM > Creative PM.
- **Geo:** Remote preferred, Boston metro hybrid acceptable, 5-days-in-office is non-negotiable no.
- **Walk-away:** $100,000/yr base. Below = auto-no.
- **Override conditions:** Anthropic specifically OR any role with $250k+/yr base — would relocate for either.
- **Stakes:** Severance window ~2026-05-11; COBRA/ACA SEP deadline 2026-07-03; pipeline target 5 quality apps/week starting weeks 3–4.

## Operating Rhythm Summary

The day is one container — **8:30 AM (post-gym) to 5:30 PM**, with a mandatory 1–2 hr break around 1–2 PM. Inside that container, building / searching / applying / outreach run in parallel; there are no fixed time-of-day slots for individual activities. The first hour (8:30–9:30 AM) is **Sacred Learning** — articles / YouTube / podcasts on AI + PM craft, no exceptions. Friday 4:30 PM is the only mandatory weekly ritual (60-min retro, self + Claude). 5:30 PM is the active-admin hard stop; informal learning bleed after is fine. Phase arc: weeks 1–2 admin-heavy, 3–4 apps hot, 5–6 interview loops + MCP ship, 7–8 offers/decisions, with Track-C building emphasized across all phases. See [[HEARTBEAT]].

## Decision Pattern Summary

Recurring decisions cluster around "what to build / how to stand out" more than "apply or skip." Real prioritization stack: **geo fit → warm intro available → company brand/size → archetype (AI > Tech > Creative) → comp above $100k → learn-and-grow potential → energy at the moment.** Auto-yes: warm-intro outreach, AI-native company recruiters, big-name companies, anything ≥$100k, learn-and-grow roles. Auto-no: cold-recruiter spam below walk-away, non-remote/non-Boston-hybrid, 5-days-in-office, bootcamps/consulting, anything <$100k. Daily kickoff is **resume yesterday's open thread** — Claude writes tomorrow's prompt at end-of-day; deep_researcher + Obsidian web clipper feed the project pipeline between sessions. **Boundary rule:** every word that hits another human's inbox is Sean's; agents draft, Sean sends. Tiebreakers stack: warm intro > bigger brand > rising-AI-co-validated-by-weekly-research > whichever ships faster. See [[USER]].

## Dependency Map Summary

Critical-path collaborators are narrow: **Mary, Matt Vitebsky, Larry Cermak.** All three are reference / referral / connection sources; the 9 P&E peers + warm-20 are broader-network and opportunistic. Tool surface is wide and intentionally redundant — no single API/model outage would immediately halt building (Anthropic + Gemini DR + local Ollama + Pi Coding Agent + LDR/SearXNG cover the same job through different paths). Machine topology: **Mac Mini = primary build station** (highest blast radius), **MacBook Pro = travel/couch + cross-over hot-backup**, **Alienware = OSS testing + creative work** (lowest blast radius for the hunt). **Single source of truth: `vault/20_projects/prj-job-hunt-2026/README.md`.** The Track-C MCP cold-start chain (name → repo → README → plan) is the most concrete current blocker for the differentiator artifact. See [[SOUL]] Part A.

## Institutional Context Summary

Working glossary: Track A/B/C, Onwards-and-Upwards, 5/4 huddle, Walk-away, Warm-20, Karpathy single-top-rec, AI/Tech/Creative PM archetypes, Sacred first hour. Comms norms: recruiters within 24h, warm-network within 48h, cold-recruiter never. Long-form on Substack, short-form on Twitter, LinkedIn amplifies both. Tone is **story-driven Sean Mode** ([[writing-voice-modes]] — Sedaris-tuned). Anti-patterns: never LinkedIn one-click apply, never "just checking in," always lead with kind + thankful + genuinely curious. Past landmines to defend against: applying too fast → burnout by week 4; all-eggs-in-one-basket; skipping company research; forgetting the post-interview thank-you. Tacit knowledge for collaborators: **Sean works best in the morning, the mid-day break is mandatory, and he'll take on any task however out-of-realm — push him toward learn-and-ship work.** See [[SOUL]] Part B.

## Active Leverage Points

- **The 8:30–9:30 AM learning hour.** Compounds across the full 8 weeks; protect it absolutely.
- **Track-C MCP server (intent-engineering).** The single-highest-leverage portable artifact per the Karpathy thesis. Ship v0 by 2026-05-25; cold-start chain is the unblocker.
- **Warm intros over cold apps.** Past wins are referral-driven; weight Mary / Matt / Larry channels above LinkedIn one-clicks.
- **The Friday retro as a forcing function.** It surfaces what worked / didn't and sets next week's headline goal — the cheapest course-correction loop in the system.
- **Substack + Twitter + LinkedIn cross-posting.** One source of truth (Substack), syndication amplifies. Currently blocked on Substack voice + build-in-public format decisions.

## Known Bottlenecks

- **Track-C MCP cold start** (no name / repo / README / plan yet) — gates the differentiator artifact.
- **Substack voice + build-in-public format decisions** — gate the public-surface track.
- **Target list of 30 companies** — gates the application volume ramp-up in weeks 3–4.
- **Agent fleet audit + Mac-Mini migration** — anything depending on MBP/Alienware-awake is too flaky for an active hunt; needs an explicit restructuring pass.
- **Gmail labeling pipeline** is unbuilt — recruiter / interview / reference / network email is undifferentiated in the inbox today.
- **Self-blocking decisions pile up** — target list, voice, YouTube yes/no, build-in-public format, next thing to build. All Sean-only calls.

## Cross-Domain Bleed

- **Life-systems → job-hunt:** finance milestones (severance, UI, COBRA SEP 2026-07-03) live in `life-systems/` but their deadlines drive job-hunt urgency.
- **Creative-studio → job-hunt:** writing-voice-modes (Sean Mode) is the tone baseline for Substack + LinkedIn; pixel-art / Remotion / 16BitFit work *is* portfolio surface for the Creative PM archetype if it comes up.
- **Job-hunt → life-systems:** energy management (the mid-day break, the 5:30 hard stop, the morning learning hour) is a life-systems concern that this domain depends on; if life-systems rhythm slips, job-hunt rhythm slips.
- **Job-hunt → claude-mastery:** the Track-C MCP server is itself a claude-mastery artifact; learnings from building it should flow back into the skill set.
