# 16Bit Fit Color Palette

Use **Shell light gray (#C4BEBB)** as the background or main UI container color.
Use **Shell darker gray (#545454)** to differentiate panels or menus.
Use **Button black (#272929)** for icons, outlines, and text contrast.
Use **A/B button color (#9A2257)** for primary buttons or call-to-action elements.
Use **Screen border green (#84D07D)** for headers or focus areas.
Implement the green screen palette (**#9BBC0F**, **#8BAC0F**, **#306230**, **#0F380F**) within game content or simulated screens.
Accent text/buttons with **#5577AA** – steely blue, echoes the “Nintendo GAME BOY™” text
