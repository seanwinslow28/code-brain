## Enhanced Notes

### Pro Platform Evolution & Market Context

- Started with research and news only pre-FTX
	- 6-8 research reports weekly cadence
	- Early industry advantage - limited legitimate competitors
	- Massive contracts signed during bull market (6-figure packages, 10-20 seats)
- Post-FTX market downturn forced right-sizing
	- Lost crypto-native clients (Animoca, protocols, traders)
	- Retained institutional/tradfi-curious clients
	- Current focus: high-level data for institutions with smaller crypto teams

### Current Client Base & Use Cases

- Primary segments:
- Key clients examples:
	- Ripple (just re-signed, increased seat count)
	- Bitwise (major data client, uses API for quarterly reports)
	- Spanish regulators (3+ years, needed staking education in January)
	- MasterCard (custom service hours project on token protection)
- $556k renewals last year on essentially unchanged platform

### Platform Strengths & Gaps

- Data by far biggest hit - flexibility to add new datasets crucial
	- Polymarket/Kalshi comparisons highly valued
	- Stablecoin analysis (Simon/Thiago’s custom work)
	- VC funding data popular but needs constant category updates
- Major limitation: API access restrictions
	- Multiple prospects would buy API if Kalshi data available
	- RWA data was added then removed
	- “I probably had 3 conversations past 4 months with people that would have bought API if we could give them Kalshi data”

### Client Pain Points & Requests

- ROI measurement challenge: “How can I tie ROI to this research report? You can’t. It’s a nice to have”
- Token Terminal-style tool needed: Simon bot upgraded with full data integration
	- Should pull news, research, data for comprehensive topic summaries
	- Example: “Give me full scope of prediction markets” with supporting data
- Modular dashboard: Clients want 3-4 key charts daily, not full platform
	- JP Morgan example: “I don’t want to log into 2 websites, just put RWA data on my dashboard”
- Better discoverability: Active clients don’t know team can do custom work

### Strategic Recommendations

- Adaptive platform concept: Homepage changes quarterly based on major narratives
	- Since 2023 would have covered: NFTs → ETFs → meme coins → Solana → RWAs → prediction markets → Hyperliquid
	- Much more manageable for tradfi audience vs crypto-native
- Positioning shift: From “nice to have” to “must have” for crypto-tradfi bridge
	- Target: Companies forced to integrate crypto (regulators, payment providers, institutions)
	- Focus on 4-5 key areas: top coins, Hyperliquid, RWAs, stablecoins, prediction markets
- UI/UX overhaul needed: Platform looks dated for $7,500/month product
- Enhanced notification system: Keyword-based alerts for relevant topics
