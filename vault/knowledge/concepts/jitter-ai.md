## Definition

A feature within Jitter that allows users to generate custom creative tools using natural language prompts, enabling effects like ripple animations or glitch transitions directly within the Jitter interface.

## Context

Relevant to Sean's creative-studio workflows and potential job-hunt-2026 portfolio production, as it allows rapid prototyping of motion design elements.

## Evidence

> Today we're launching Jitter AI: an unlimited set of creative tools you can build yourself with just a prompt.

> Simply describe the creative tool you need (such as a fluted glass background, a ripple effect, a glitch transition, a 3D rotation, or a card flip), and Jitter will build it for you, right inside the interface you already know.

## Examples

- Add a dithering effect
- Create an animation with a pixelated effect
