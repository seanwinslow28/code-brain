# AI Animation Technique: NB2 + Seedance 2.0 + Procreate

After working on my 2D frame by frame animation for my portfolio, I realized that Nano Banana 2 gives amazing results for the specific style you're trying to achieve. The issue is when you bring it into a video generation model, the character/environments have a tendency to morph and look odd in some frames. 
I then thought, "Well, what if we extract the video model frames to get the motion down and then bring into NB2 for cleanup through frame regenerations in the style I'm trying to achieve with a reference or 2 and Procreate for timing and more clean up (the human in the loop)". 
Today is April 13th, 2026. Seedance 2.0 is available in the U.S and the video results are perfect. If we combine that with NB2 and Procreate, we can make something pretty special with the help of AI. This can also work for the sprite sheets.

## Workflow

1. **Seedance 2.0** — Generate motion/video with correct movement
2. **Extract frames** — Pull individual frames from video output
3. **Nano Banana 2** — Regenerate each frame in target art style using reference images
4. **Procreate** — Human-in-the-loop timing adjustments and final cleanup

## Applications

- Portfolio animated short
- [[prj-16bitfit]] sprite sheets
