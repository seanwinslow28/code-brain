## Sample Obsidian Note Structure for Design Patterns

```markdown
---
type: design-pattern
category: animation
tags: [micro-interaction, button, feedback]
use-case: button-click-feedback
framework: react
library: framer-motion
---

# Button Click Feedback Animation

## When to Use
- Primary action buttons
- Form submissions
- Any click that triggers an action

## Code
\`\`\`tsx
<motion.button
  whileTap={{ scale: 0.95 }}
  whileHover={{ scale: 1.02 }}
  transition={{ type: "spring", stiffness: 400, damping: 17 }}
>
  Click me
</motion.button>
\`\`\`

## Variations
- [[Subtle Button Feedback]] - for secondary actions
- [[Loading Button State]] - when action takes time

## See Also
- [[Framer Motion Basics]]
- [[Button Design Patterns]]
```

This structure lets Claude Code query your vault for "button animation patterns" and get immediately usable code.
