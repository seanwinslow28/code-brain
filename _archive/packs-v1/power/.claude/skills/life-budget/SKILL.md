# Life Budget Skill

## When to Use

Use this skill when:
- Creating budget spreadsheets or apps
- Tracking expenses and income
- Analyzing spending patterns
- Setting up financial automation
- Planning savings goals

## Examples

**Example 1: Expense tracking setup**
```
User: "Create a budget tracker for monthly expenses"
Claude: [Uses life-budget skill] Creating budget tracker with categories:

**Income:**
- Salary: $X
- Side income: $Y

**Expenses:**
- Housing (rent/mortgage): $X (30% of income)
- Food: $Y (15% of income)
- Transportation: $Z (10% of income)
- Utilities: $A
- Savings: $B (20% target)
- Discretionary: $C

**Tracking:**
- Daily expense logging
- Weekly budget review
- Monthly analysis and adjustments
```

**Example 2: Financial automation**
```
User: "Set up automation to track credit card expenses"
Claude: [Uses life-budget skill] Automation setup:

1. Email parsing: Extract transactions from bank emails
2. Categorization: Auto-categorize by merchant/description
3. Budget alerts: Notify when category exceeds limit
4. Monthly reports: Generate spending analysis
5. Goal tracking: Monitor progress toward savings goals

Script structure:
- Parse email/CSV exports
- Match transactions to categories
- Update budget spreadsheet
- Generate alerts and reports
```

## Budget Features

This skill helps with:
